### System requirements:
* Git
* Make
* Docker

### Application setup:
```sh
$ git clone https://github.com/algins/devops-for-developers-project-74.git && cd devops-for-developers-project-74
$ make setup
```

### Start the development server:
```sh
$ make start
```

### Run tests:
```sh
$ make test
```
